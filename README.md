SmartTech Company Case Study (Modified)
SmartTech is a software company that works with multiple clients and develops different projects.
The company has several departments.
Each department works on multiple projects.
Every employee:
- Works in exactly one department.
- Works on exactly one project.
- The company records the number of hours the employee worked on their assigned project.
Each project:
- Belongs to one department.
- Is developed for one client.
- Has a specific budget.
- Can have multiple employees assigned to it.
Each client:
- Can request multiple projects.
- Has a name and country.
The company wants to store the following information:
- Departments (name and location)
- Employees (name, age, salary, hire date, status, hours worked)
- Clients (name and country)
- Projects (name and budget)
